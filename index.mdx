## Introduction

Eco Executor is a plugin that almost replicates real roblox executors made for roblox developers to pentest/debug their game.

## Installation

To install the plugin, go to the [Eco Executor Github repo](https://github.com/Overhault-Experiences/Eco-Executor) and then go to the **releases**, and download the **latest one**.
